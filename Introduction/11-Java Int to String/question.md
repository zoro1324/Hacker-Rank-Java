# Java Int to String

You are given an integer `n`; your task is to convert it into a string `s`.

Complete the partially completed code in the editor. If your code successfully converts `n` into the string `s`, the program will print **"Good job"**; otherwise it will print **"Wrong answer"**.

---

## Constraints

- `n` is an integer in the range **-100** to **100**, inclusive.

## Input Format

A single integer `n`.

## Output Format

Print **"Good job"** if the conversion is successful; otherwise print **"Wrong answer"**.

## Sample Input

```
100
```

## Sample Output

```
Good job
```
